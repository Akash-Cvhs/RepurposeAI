# Molecule Image Storage

## ✅ Images Are Saved

Molecular structure images are now automatically saved to disk when the drug_analyzer_agent runs.

## Storage Location

**Directory:** `backend/archives/molecule_images/`

**Naming Convention:** `{DrugName}_{Timestamp}.png`

**Example:** `Donepezil_20260323_170011.png`

## Image Details

- **Format:** PNG
- **Size:** 400x400 pixels
- **Generated by:** RDKit `Draw.MolToImage()`
- **Source:** SMILES notation from drug database

## How It Works

When the drug_analyzer_agent analyzes a drug:

1. **Fetches SMILES** notation from database
2. **Generates 2D structure** using RDKit
3. **Saves to disk** at `archives/molecule_images/`
4. **Returns both:**
   - File path (for disk access)
   - Base64 data (for embedding in reports/APIs)

## Example Output

```python
{
    "base64": "data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAA...",
    "file_path": "C:/Users/.../backend/archives/molecule_images/Donepezil_20260323_170011.png",
    "filename": "Donepezil_20260323_170011.png"
}
```

## Usage in Workflow

### Disease-Based Search
When searching drugs by disease (e.g., "Alzheimer's drugs"):
- Finds 7 candidates
- Generates image for each (top 5)
- Saves to: `Donepezil_*.png`, `Galantamine_*.png`, etc.

### Single Drug Analysis
When analyzing a specific drug:
- Generates one image
- Saves with drug name + timestamp

## Accessing Images

### From Code
```python
from agents.drug_analyzer_agent import DrugAnalyzerAgent

agent = DrugAnalyzerAgent()
state = {"query": "Analyze Donepezil", "molecule": "Donepezil"}
result = await agent.analyze_drug(state)

# Get image info
analysis = result.get("single_drug_analysis")
# Contains: [Image saved to: .../Donepezil_20260323_170011.png]
```

### From File System
```bash
cd backend/archives/molecule_images
ls *.png
```

### From API Response
The orchestrator returns the full state including image data:
```json
{
  "drug_analysis": "...",
  "candidate_analyses": [
    {
      "drug_name": "Donepezil",
      "structure_image": {
        "base64": "data:image/png;base64,...",
        "file_path": ".../Donepezil_20260323_170011.png",
        "filename": "Donepezil_20260323_170011.png"
      }
    }
  ]
}
```

## Image Cleanup

Images accumulate over time. To clean up old images:

```bash
# Delete images older than 7 days
find backend/archives/molecule_images -name "*.png" -mtime +7 -delete

# Or delete all
rm backend/archives/molecule_images/*.png
```

## Configuration

Images are saved by default. To disable disk saving:

```python
# In drug_analyzer_agent.py
image_data = self._generate_molecule_image(smiles, drug_name, save_to_disk=False)
```

This will only return base64 data without saving to disk.

## Integration with Reports

The report generator can:
1. **Reference file path** in markdown reports
2. **Embed base64** in HTML/PDF reports
3. **Link to images** in web interfaces

Example in report:
```markdown
## Molecular Structure
[Image saved to: .../Donepezil_20260323_170011.png]
```

Or embedded:
```html
<img src="data:image/png;base64,iVBORw0KGgo..." />
```
